# 深度学习水果和蔬菜识别系统 python88

## 项目概述

本项目是基于深度学习技术的的水果和蔬菜识别系统，采用了mobilelnet模型，能够识别36种常见的水果和蔬菜。

## 技术栈

- 开发环境：Pycharm+Anaconda
- 编程语言：Python 3.9
- 依赖库：opencv_python, PyQt5

## 功能模块

### 核心功能

- 实时蔬菜识别：通过选择图片，系统能够快速并准确地识别出水果和蔬菜。

## 系统角色

- 用户：可以通过UI界面选择图片进行水果和蔬菜的识别。

## 运行环境

- Python版本：3.9
- 必要的库：opencv_python, PyQt5

## 部署步骤

1. 确保Python环境为3.9版本。
2. 安装必要的库：`opencv_python` 和 `PyQt5`。
